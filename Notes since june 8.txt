We could use ADC multixplexers. The drawback w/ multiplexers is that we can only recognize one input at a time. Let's think about this, is this okay?

Therefore, it is okay! We don't even need a new PCB because the CD74HCx4067 multiplexer takes only the following inputs:
- power
- ground
- enable
- output to an ADC pin on microcontroller
- output to select pins on microcontroller

which we can Frankenstein easily enough


Note: the multiplexer device has a maxmimum source/sink current of 25mA. How do we control for that.\
Solution: recall the functionality of the open drain device. When enough gate voltage is applied, device is shorted, no current. When not enough gate voltage is applied, current determined by pullup resistor. If we use 5kohms, then 3.3V/5kohms = 0.66mA, far less than limit
